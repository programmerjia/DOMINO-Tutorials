# Installation
The DOMINO package is developed based on the pytorch framework and can be implemented on both GPU and CPU. We recommend running the package on GPU.  We provide two ways to install the package of DOMINO as follow.

## 1.Python

```bash
pip install domino-spatial
```

## 2.Conda

For convenience, we suggest using a separate conda environment for running DOMINO. Please ensure annaconda3 is installed.

Create conda environment and install DOMINO package.

```bash
#create an environment called DOMINO
conda create -n DOMINO python=3.8

#activate your environment
conda activate DOMINO

#install package

pip install domino-spatial
```
